# Ghi Chú Âm Thanh

Ghi Chú Âm Thanh là một blog âm nhạc được xây dựng bằng Next.js, tập trung vào playlist, review album, live session và những câu chuyện xoay quanh văn hóa nghe hiện đại. Dự án hướng đến trải nghiệm đọc gọn gàng, rõ ràng và có cá tính, phù hợp cho một tạp chí âm nhạc cá nhân bằng tiếng Việt.
